# PricePulse - Amazon Price Tracker

![PricePulse Dashboard](image_ad245f.png)

A real-time Amazon product price tracker. This project uses a Python and Flask backend to scrape product data and a simple HTML/JavaScript frontend to display it in a live dashboard.

---

## Features

* Tracks product prices from both **Amazon.com** (USD) and **Amazon.in** (INR).
* Converts all prices to INR for a unified dashboard.
* Real-time tracking: add new products directly from the UI.
* Live-updating price history charts for each product.
* Saves product data to a local `product_data.json` file.
